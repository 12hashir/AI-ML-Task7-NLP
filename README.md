# AI & ML Course - Task 7: Natural Language Processing Basics

## Overview
This repository contains the solution for Task 7: NLP Basics. The task involves building an NLP pipeline on a custom coretech_feedback.csv dataset.

## Files Included
- Task7_NLP.ipynb : Complete Colab notebook with all NLP steps
- coretech_feedback.csv : Dataset with 30+ feedback entries

## NLP Pipeline Implemented
1.  *Text Cleaning:* Lowercase, remove punctuation and stopwords
2.  *Tokenization & Lemmatization:* NLTK used
3.  *TF-IDF Vectorization:* Feature extraction 
4.  *Visualization:* Sentiment Distribution Bar Chart & Word Cloud
5.  *Modeling:* Multinomial Naive Bayes for Sentiment Classification
